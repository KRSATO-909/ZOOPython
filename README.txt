Информационная система зоопарка
Настольное приложение на Python для учёта:
- Животных
- Вольеров
- Сотрудников
- Ветеринарных осмотров
- Кормления

Для работы с приложением необходим предустановленный Python версии не ниже 3.9
Чтобы запустить приложение необходимо:
1. Открыть командую строку (Win + r ; cmd)
2. Прописать в командой строке путь к папке с содержимым проекта, пример: cd C:\Users\Krsato\Documents\ZOOPython
3. Запустить виртуальную среду Python с помощью команды: venv\\Scripts\\activate
4. Запустить приложение в виртуальной среде через команду: python main.py

В дальнейшем работа будет продолжаться в десктопном приложении, которое связано с базой данных, при желании ее можно удалить (очиститься все данные бд), если ее нет - она создается автоматически.
